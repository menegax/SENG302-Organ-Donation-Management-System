# User Manual
last update 03/03/2018

### Requirements

* JDK 1.8


### Setup

1. Download the .jar file
2. Double click the application to begin 
3. Type "quit" into the command line to quit the application
4. Use -h for help

Usage: odms [-h]<br>
  -h, --help                  Displays this help message and quits.<br>
